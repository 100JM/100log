깃허브에 TIL 작성을 시작하면서 액션을 통해 README를 자동으로 작성해주는 방법이 있다길래 사용 해보는 도중 push를 했을때 아래와 같은 오류를 만났다...   

![](https://velog.velcdn.com/images/100-100/post/f2a5488c-c703-4875-b424-3d5394e089db/image.png)   

이리저리 검색해본 결과(권한, 토큰 문제 등) 여러 상황이 있는데 나같은 경우 Workflow 권한 문제였다.

참고 : https://stackoverflow.com/questions/73687176/permission-denied-to-github-actionsbot-the-requested-url-returned-error-403

해당 레포지토리 - settings - actions(General) - Workflow permissions 부분에서 Read and write로 권한을 설정해주니 해결되었다 !
