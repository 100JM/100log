🔍 기능
---
1. 주요 기능
- 전체 & 카테고리별 게시글 목록
- 게시글 상세 페이지
- 게시글 상세 내용 목차(사이드 바)
- 댓글 기능
- 자기소개 페이지(about)   

2. 부가 기능
- 최근 게시글 고정
- 게시글 목차(사이드 바)
- 라이트 & 다크 모드
- 현 위치 날씨 정보

🛠️ 기술 스택
---
- Framework: Next.js(14)
- Design: Tailwind css & Remix icon
- 게시글 관련: MDX(작성) & gray-matter, next-mdx-remote(파싱)
- 댓글: Giscus
- 날씨 정보: Openweathermap api
- 상태 관리: SWR & Zustand
